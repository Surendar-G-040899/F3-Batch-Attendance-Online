       STEPS TO EXECUTE THE PROJECT:
STEP 1: Open OBS Studio.
STEP 2: Start a Google Meet.
STEP 3: In OBS Studio, Add Google meet in Windows capture.
STEP 4: Open Command Prompt.
            STEP 4.1: Type the following Commands..
            STEP 4.2: cd Desktop
            STEP 4.3: cd Attendance_proj
            STEP 4.4: Flask run
STEP 5: Open Web Browser and add the obtained URL.
STEP 6: Enter the duration in minutes and the attendees to monitor.
STEP 7: Attendance marking will start in cmd.
STEP 8: The output screen with attendance marked will be displayed.
STEP 9: To change the attendance manually, it can be downloaded.
